# homebrew-tap

Homebrew tap for [Prem5123](https://github.com/Prem5123)'s projects.

## Installing

```bash
brew install Prem5123/tap/multica-autodev
```

Or, if you prefer:

```bash
brew tap Prem5123/tap
brew install multica-autodev
```

## Available formulae

| Formula | Description |
|---|---|
| `multica-autodev` | Continuous improvement sidecar for [Multica](https://github.com/multica-ai/multica). |

## How this tap is updated

Formulae here are auto-published by [GoReleaser](https://goreleaser.com)
when a `v*` tag is pushed to the source repository. Do not edit
`Formula/*.rb` by hand — those changes will be overwritten on the next
release.
